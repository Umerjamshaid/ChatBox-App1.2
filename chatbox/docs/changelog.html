Changelog

Stay updated with the latest features, improvements, and bug fixes in ChatBox.

Version 2.0.0
December 15, 2024

Added

  • End-to-End Encryption: Complete message encryption with ECDH key exchange
  • Database Integration: SQLite database with automatic migrations and performance monitoring
  • Chunked Storage: SharedPreferences chunking for large data handling
  • Type Safety: Complete model classes replacing Map usage
  • Voice Commands: Advanced speech-to-text with multi-language support

Changed

  • API Integration: Replaced mock data with real Stream Chat API calls
  • UI Framework: Migrated to modern Flutter widgets with improved animations
  • Performance: Optimized serialization/deserialization with custom codecs

Fixed

  • Reaction API: Fixed deprecated reactionCounts usage
  • Unread Count: Corrected user ID comparison in message provider
  • Memory Leaks: Fixed stream controller disposal issues
Version 1.5.0
October 28, 2024

Added

  • Offline Support: Message queuing and automatic sync when back online
  • Message Reactions: Emoji reactions with real-time updates
  • Voice Messages: Record and send voice notes with waveform display
  • Group Management: Create and manage group conversations

Changed

  • UI Polish: Improved animations and transitions throughout the app
  • Performance: Optimized message list rendering with virtualization
Version 1.2.0
August 15, 2024

Added

  • Real-time Chat: WebSocket-based instant messaging
  • Presence Indicators: Online/offline status for all users
  • Typing Indicators: See when others are typing
  • Message Search: Full-text search across all messages

Fixed

  • Connection Issues: Improved WebSocket reconnection logic
  • Memory Usage: Reduced memory footprint for large chat histories
Version 1.0.0
June 1, 2024

Added

  • Initial Release: Basic messaging functionality with Firebase integration
  • User Authentication: Email/password and Google sign-in
  • Channel System: Public and private conversation channels
  • Media Sharing: Image and file sharing capabilities
  • Push Notifications: Firebase Cloud Messaging integration
