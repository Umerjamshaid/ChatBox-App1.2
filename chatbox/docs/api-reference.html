API Reference

Complete API documentation for ChatBox services and components.

Authentication API

POST /auth/login

Authenticate user with email and password

Parameters

Name Type Required Description
email string Yes User's email address
password string Yes User's password

Response

{ "success": true, "user": { "id": "user_123", "email": "user@example.com", "name": "John Doe", "token": "jwt_token_here" } }
POST /auth/register

Register a new user account

Parameters

Name Type Required Description
email string Yes User's email address
password string Yes User's password (min 8 characters)
name string Yes User's display name

Messages API

GET /channels/{channelId}/messages

Retrieve messages for a specific channel

Parameters

Name Type Required Description
limit integer No Number of messages to retrieve (default: 50, max: 100)
before string No Message ID to paginate before

Response

{ "messages": [ { "id": "msg_123", "text": "Hello world!", "userId": "user_456", "userName": "John Doe", "createdAt": "2024-01-15T10:30:00Z", "status": "sent", "reactions": [] } ], "hasMore": true }
POST /channels/{channelId}/messages

Send a new message to a channel

Parameters

Name Type Required Description
text string Yes Message content
attachments array No Array of file attachments

Channels API

GET /channels

Get user's channels

Response

{ "channels": [ { "id": "channel_123", "name": "General", "type": "public", "memberCount": 25, "lastMessage": { "text": "Welcome to the team!", "userName": "Admin", "createdAt": "2024-01-15T09:00:00Z" } } ] }
POST /channels

Create a new channel

Parameters

Name Type Required Description
name string Yes Channel name
type string No Channel type (public/private)
members array No Array of user IDs to invite

Voice API

POST /voice/transcribe

Transcribe audio to text

Parameters

Name Type Required Description
audio file Yes Audio file (WAV, MP3, M4A)
language string No Language code (default: auto)

Response

{ "text": "Hello, how are you today?", "confidence": 0.95, "language": "en-US", "duration": 2.3 }
